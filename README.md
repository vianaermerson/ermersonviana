\# 🎀 Enquete do Chá Revelação



Sistema completo de votação para Chá Revelação com interface moderna, animações e painel administrativo.



\## ✨ Funcionalidades



\- ✅ Votação para MENINO ou MENINA

\- ✅ Prevenção de votos duplicados (LocalStorage + IP)

\- ✅ Resultados em tempo real com barras animadas

\- ✅ Efeito de confetes ao votar

\- ✅ Contagem regressiva para o evento

\- ✅ Compartilhamento WhatsApp/Twitter

\- ✅ QR Code para acesso rápido

\- ✅ Painel administrativo protegido por senha

\- ✅ Design responsivo (celular/tablet/desktop)
